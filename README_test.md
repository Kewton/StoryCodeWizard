# Next.js チャットアプリケーション開発要求仕様書

## 概要
Next.jsを用いて、Teamsのようなチャットインターフェースを持つwebアプリケーションを開発する。人間とAIを問わず、不特定多数のユーザーとのコミュニケーションを可能とする。

## UIコンポーネント構成
- チャットウィンドウ
  - メッセージ一覧
    - メッセージバブル（送信者名、メッセージ本文、タイムスタンプ）
  - メッセージ入力フォーム
    - テキスト入力欄
    - 送信ボタン
- サイドバー
  - ユーザー一覧
  - ルーム一覧
- ヘッダー
  - アプリケーションタイトル
  - ユーザー設定メニュー
